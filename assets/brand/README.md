# Optional Official Brand Assets

Place any user-provided official FIFA / World Cup logo files here if you decide to use them as visual references or accents.

## Important

- Do not commit downloaded official logos unless you are comfortable with the rights/trademark implications for your use case.
- The app should run without anything in this folder.
- Any official marks should be treated as optional brand accents, not hard dependencies.
- Do not trace, recreate, or generate lookalike versions of protected marks.
- Keep filenames descriptive, for example:
  - `fifa-world-cup-2026-logo.png`
  - `fifa-world-cup-2026-logo.svg`
  - `host-city-mark-reference.png`

## Recommended Use

Use official assets sparingly:

- Small header accent.
- Rules/about page reference.
- Draft night splash screen.
- Optional background watermark at very low opacity.

Avoid building the entire UI around official marks. The core visual system should come from soccer textures, flags, scoreboards, pitch lines, stadium light, and league-specific identity so it still feels strong without protected assets.
