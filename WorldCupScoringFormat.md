# World Cup Fantasy Scoring Format

## Game Concept

This is a fantasy football-style game for the 2026 FIFA World Cup. Instead of drafting individual players, each participant drafts entire countries onto their roster.

The goal is to build a roster of teams that performs well across the full tournament, not simply to pick the eventual World Cup champion. Points are earned through match results, goals, defense, discipline, and a few small tournament bonuses.

The format is designed for 8 to 12 participants.

## Draft Format

The 2026 World Cup has 48 countries.

Recommended roster sizes:

| Participants | Countries Per Participant |
| --- | ---: |
| 8 | 6 |
| 9 | 5, with 3 extra countries assigned by reverse draft order |
| 10 | 4, with 8 extra countries assigned by reverse draft order |
| 11 | 4, with 4 extra countries assigned by reverse draft order |
| 12 | 4 |

A snake draft is the simplest recommended format.

For leagues that want more strategy and fairness, an auction draft can also work well.

## Core Scoring

These rules apply to all matches unless otherwise noted.

| Event | Points |
| --- | ---: |
| Win | +1 |
| Draw | +0.5 |
| Goal scored | +0.5 |
| Goal allowed | -0.25 |
| Clean sheet | +0.5 |
| Red card | -1 |

Draw points only apply during the group stage. Knockout matches do not award draw points because every knockout match has a winner.

Penalty shootout goals do not count as goals scored.

## Group Stage Bonuses

| Event | Points |
| --- | ---: |
| Qualify for knockout stage | +0.5 |
| Win group | +1 |

The qualification bonus is intentionally small because 32 of the 48 teams advance to the knockout stage.

## Tournament Bonus

| Event | Points |
| --- | ---: |
| Win the World Cup | +2 |

There are no round-by-round knockout advancement bonuses.

This keeps the eventual champion valuable, but prevents the game from becoming a simple contest of who drafted the champion.

## Clean Sheet Definition

A clean sheet means a country allows zero goals in a match.

Examples:

| Result | Clean Sheet Awarded? |
| --- | --- |
| 1-0 win | Winning team gets a clean sheet |
| 0-0 draw | Both teams get a clean sheet |
| 2-1 win | Neither team gets a clean sheet |

## Knockout Stage Notes

In the 2026 format, the World Cup champion must win 5 knockout matches:

1. Round of 32
2. Round of 16
3. Quarterfinal
4. Semifinal
5. Final

With this scoring format, the champion earns:

| Source | Points |
| --- | ---: |
| 5 knockout wins | +5 |
| World Cup champion bonus | +2 |
| Total result-based knockout value | +7 |

The champion will also earn or lose points from goals scored, goals allowed, clean sheets, and red cards.

## Example Match Scoring

Example: Team A beats Team B, 2-1.

Team A:

| Event | Points |
| --- | ---: |
| Win | +1 |
| 2 goals scored | +1 |
| 1 goal allowed | -0.25 |
| Total | +1.75 |

Team B:

| Event | Points |
| --- | ---: |
| 1 goal scored | +0.5 |
| 2 goals allowed | -0.5 |
| Total | 0 |

Example: Team A and Team B draw, 0-0, during the group stage.

Each team:

| Event | Points |
| --- | ---: |
| Draw | +0.5 |
| Clean sheet | +0.5 |
| Total | +1 |

## Design Goals

This scoring system is built to:

- Keep the standings close across 8 to 12 participants.
- Make every group-stage match matter.
- Reward good attacking teams without over-punishing teams in high-scoring matches.
- Make clean sheets useful but not dominant.
- Keep underdogs relevant.
- Make the World Cup champion valuable without making the champion pick automatically decide the league.
- Avoid overly complicated penalty shootout rules.
